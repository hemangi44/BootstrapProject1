Student Management System Project In Python

This is a simple Student Management System project is written in Python and MySQL. The project file contains a python script (menu.py). This is a simple console based system which is very easy to understand and use. Talking about the system, it contains basic functions which include Add students,add subject-wise marks, view students, update student's data and remove the student. It also calculates overall percentage of entered marks. In this mini project, there is no such login system. It is too easy to use, one can check the total student’s record.

Another feature of this mini project is it makes use of data visualization tool Matplotlib. With the help of this module the user can view the chart in bar format in order to check all stutdents percentage.

While adding the students, the user only has to enter his/her name and subject-wise marks then the system adds the record and displays to the user along with the percentage. And the user can view all these students lists from the view section. The user can also update the student's data if roll no. is provided. Moreover the user can remove student's data permanently if the particular roll no. is provided. There is database connection used in this mini project to save user’s data permanently.

Features

  1. Insert
  2. View Details
  3. View Chart
  4. Update Details
  5. Remove Data
