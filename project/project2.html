Using the 960 Grid System as a Design Framework

The 960 Grid System is simply a way to lay out websites using a grid that is 960 pixels wide. The reason it’s 960 pixels wide is because the number 960 makes for a lot of clean divisions utilizing whole numbers when factoring in column widths and margins. And it fits nicely on the majority of screens.

This project is based on 12-column version where the narrowest column is 60 pixels wide. So the available column widths are: 60, 140, 220, 300, 380, 460, 540, 620, 700, 780, 860 and 940.

This project about Interior Design Planner which has been developed on HTML and CSS. The main objective of developing this project was to create static website for Interior Designing, from which the user can get information about of interior designing such as products,services offered,themes, contact details, company information,etc.
